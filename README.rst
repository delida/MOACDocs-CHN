墨客开发中文文档
===============

需要安装以下工具来产生最终的HTML格式文件:

MAC 安装:

1. Install sphinx：

 brew install sphinx-doc

2. Install the sphinx_rtd_theme:

sudo pip install sphinx_rtd_theme

3. Enter Docs directory, and run Makefile

cd Doc
make html
